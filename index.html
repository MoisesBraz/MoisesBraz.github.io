Sou Estudante - Moisés

Chamo-me Moisés Matos Braz, estudo na ESTGA - Universidade de Aveiro, na Lincenciatura de Engenharia Informática Aplicada.
Conclui o Curso Técnico Superior de Programação de Sistemas de Informação, na ESTGA - Universidade de Aveiro.

Trabalhei durante mais de ano como churrasqueiro na firma churrascaria VIASA, de Conde, Tuca & Quinteiro.
Ganhei experiências mais aprofundadas na área de IoT durante o estágio curricular, realizado no Curso Superior Profissional de Programação de Sistemas de Informação.

Projetos principais que foram desenvolvidos durante o estágio curricular:
1º Criação de um website com dashboard para visualização das linhas de operação com as equipas de trabalho e turnos, utilizando Node.js, Express e Passport.
2º Implementação de uma página web com funcionalidades CRUD, autenticação de utilizadores. A aplicação inclui um formulário, com o "id" da linha e as datas que a linha operou, para obter linhas de operação/OF e realizar testes predefinidos do sistema, para a verificação da OEE das linhas de operação.
3º Criação de um Web Support Chat automatizado com PHP, HTML, CSS e JavaScript, integrando uma solução de chat bot AI.
4º Criação de uma API, com diversas funcionalidades entre as quais: inserir novo conteúdo na base de dados vetorial, retirar os dados dos clientes, enviar perguntas ao GPT e devolver respostas conforme as perguntas realizadas e os dados inseridos na base de dados vetorial. Foi utiliza a ferramenta LangChain e a API da OpenAI para gerar respostas, com base nos dados inseridos, para a página Web Support Chat.

Tenho alguns conhecimentos na área da informática: Java, Python, PHP, HTML, CSS, JavaScript, TypeScript, SQL e PostgreSQL, MariaDB, MongoDB, Node.js, Django, REST framework, LangChain.
Considero-me uma pessoa proativa e que gosta de trabalhar em equipa, contribuindo desta forma para o crescimento do grupo.
